# SPASM-V1.0---Student-s-Personal-Assistant-Synopsis-Maker-V1.0
A deterministic, domain-aware, hierarchical text compression system that preserves structural meaning by isolating invariant anchors (equations, chapters, images) and reducing only contextual redundancy.
Thats fancy for SPASM takes your whole text book or the chapters and reduces it to keywords or the most imortant words.
You can later build up your own precis or a simple paragraph for it.
If in case you want to even avoid that read the readme file till the end there are hints how to even simplify that process.
Ok for not so smart students just copy paste the keywords as input to some LLM instance and request a human readable paragraph.
